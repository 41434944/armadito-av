UHURU-MODULES(3)
================
:doctype: manpage

NOM
---
uhuru-modules - modules de recherche de malwares


SYNOPSIS
--------

enum uhuru_mod_status {
  UHURUMOD_OK,
  UHURUMOD_INIT_ERROR,
};

enum uhuru_status {
  UHURU_CLEAN,
  UHURU_MALWARE,
  UHURU_EINVAL,
  UHURU_IERROR,
  UHURU_UNKNOWN_FILE_TYPE,
};

typedef enum uhuru_mod_status (*uhuru_mod_init_t)(struct uhuru_module *mod);

typedef enum uhuru_status (*uhuru_mod_scan)(const char *path, void *data);

typedef enum uhuru_mod_status (*uhuru_mod_close)(struct uhuru_module *module);


DESCRIPTION
-----------
Les modules de recherche sont des objets chargés par la fonction *dlopen*(3). Ils implémentent des fonctions qui sont appelées par la librairie *libuhuru*(3) pour réaliser la recherche de malwares dans des fichiers.

Les modules de recherche sont chargés dynamiquement à partir des fichiers .so situés dans le répertoire */usr/lib/libuhuru/modules* (ce répertoire peut changer en fonction de l'installation). Chaque module exporte certaines fonctions qui seront utilisées par la librairie *libuhuru*. Pour plus d'informations, se référer à *uhuru-modules*(3).

Un module de recherche doit définir les fonctions suivantes (MODULE désigne le nom du module, obtenu à partir du nom du fichier):

*MODULE_init*::
Fonction d'initialisation. Alloue la structure privée du module, initialise le module, retourne la structure allouée via 'pdata'.

*MODULE_scan*::
Fonction de recherche. Utilise la structure passée via 'pdata' (structure allouée par 'MODULE_init'); réalise la recherche et retourne un statut indiquant si le fichier contient ou non un malware.

*MODULE_close*::
Fonction de dé-initialisation. Libère la structure privée du module, dé-initialise le module.
